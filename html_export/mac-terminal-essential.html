Foundation · Movement
Navigation & Paths
Learning how to move around your folders and find where you are.
STEP 01

Finding Your Location (pwd)

Before you move, you need to know where you are. The pwd command prints the 'Working Directory' (your current folder).

pwd
💡 Tip: In Terminal, your 'Home' folder is often represented by the tilde (~) symbol.
STEP 02

Listing Files (ls)

To see what's inside your current folder, use ls. You can add flags like -a to see hidden files or -l for details.

ls -la
⚠️ Warning: Hidden files in macOS usually start with a dot (.), like .bash_profile or .zshrc.
STEP 03

Changing Directories (cd)

Use cd followed by a path to move. Use cd .. to move up one level or cd ~ to go home.

cd Desktop
cd ..

✅ Section 01 — Checklist

  • ✓ Used pwd to confirm current location
  • ✓ Listed all files including hidden ones
  • ✓ Successfully navigated into and out of a folder
02
Interaction · Files
File & Folder Management
Creating, moving, and deleting items without a mouse.
STEP 01

Creating Folders (mkdir)

Create a new directory instantly with mkdir. If the name has spaces, wrap it in quotes.

mkdir "My Projects"
💡 Tip: You can create multiple folders at once: mkdir folder1 folder2 folder3
STEP 02

Creating Files (touch)

The touch command is the fastest way to create a blank text or code file.

touch script.py
⚠️ Warning: Be careful with file extensions; Terminal won't stop you from naming a file something nonsensical.
STEP 03

Removing Items (rm)

Delete files with rm. To delete a folder and everything inside it, use rm -rf.

rm file.txt
rm -rf old_folder
⚠️ Warning: Warning: The Terminal has no 'Trash' can. Deletions are permanent and immediate!

✅ Section 02 — Checklist

  • ✓ Created a new directory for practice
  • ✓ Created a dummy file using touch
  • ✓ Safely deleted the practice files
03
Efficiency · Speed
Shortcuts & Productivity
Pro-tips to make the terminal experience much faster.
STEP 01

Clearing the View

If your screen gets too cluttered, use clear to start with a fresh blank slate.

clear
💡 Tip: You can also press Cmd + K on macOS to wipe the scrollback buffer entirely.
STEP 02

Tab Completion & History

Start typing a filename and press Tab to autocomplete it. Use the Up Arrow to cycle through previous commands.

# Type 'cd Des' then hit Tab
cd Desktop/

Keyboard Shortcuts

TabAutocomplete file/folder names
Up / Down ArrowCycle through command history
Ctrl + CAbort/Cancel the current running command
Ctrl + A / EMove cursor to Start / End of the line
